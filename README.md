## 📌 Project Overview
**Blueprints** is a collaborative, full-stack web application built using the Django framework. Developed as part of a comprehensive software engineering module, the project emphasizes agile team collaboration, robust backend architecture, and a highly polished user experience. 

The application utilizes Python for server-side logic and database management, paired with modern frontend practices to deliver a responsive and secure interface.

---

## My Role & Contributions
As a core contributor to this group project, I balanced technical full-stack development with project leadership and design oversight. 

My primary responsibilities included:
* **Project Organization & Leadership:** Took the initiative to drive the project's management framework. I set up the team's Slack workspace for streamlined communication, designed our internal organizational materials, and created a comprehensive Gantt chart to map out development phases and ensure all milestones were met on schedule.
* **UX/UI Design:** Spearheaded the frontend vision by creating complete, high-fidelity wireframes and interactive prototypes using **Figma**, translating them into a clean, intuitive, and responsive user interface.
* **Full-Stack Development:** Contributed actively to the codebase, bridging the gap between database models, Python business logic, and frontend Django templates.
